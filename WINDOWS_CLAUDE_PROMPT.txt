## 刚完成的功能（4.7 WebView Bridge）
在日历/费用 WebView 页面点击条目，自动跳转到聊天 Tab 并预填上下文消息。

**实现方式：**
- 新建 TallyJsBridge.kt（JS ↔ Android 桥接）
- CalendarView 和 ExpenseView 的 item 添加 onClick handler
- 调用 Android.navigateToChat(message)
- ChatViewModel.prefillInput() 填充输入框
- navController 自动导航到聊天 Tab

**APK 位置：** C:\Users\realw\Desktop\tally.apk

## 如果遇到问题

### 快速排查
```
# 查看实时日志（WebView 相关）
adb logcat | grep -i webview

# 查看 JS 错误
adb logcat | grep -i console

# 查看导航错误
adb logcat | grep -i "navgraph\|bridge"

# 完整 logcat 导出
adb logcat > logcat.txt
```

### 常见问题
| 问题 | 排查 |
|------|------|
| 点击无反应 | Bridge 是否注册？Android 对象是否存在？ |
| 输入框为空 | prefillInput 是否被调用？ViewModel 共享是否成功？ |
| 导航不跳转 | navController.navigate 是否出错？ |
| 网络错误 | adb reverse 是否成功？后端是否运行？ |
